# PechPech-BOT
PechPech Bot is a telegram bot based on pyrogram
for sending whisper messages in telegram chats.

## Requirements 
- 🐍 Python [3.8 or Above] 
- 📚 Pyrogram [Library]
- 📚 Redis [Library]
- 🗄 Redis Server [Database]

## Run
first run this to collect pechpech
